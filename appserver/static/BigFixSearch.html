HCL BigFix App for Splunk



Welcome to the HCL BigFix App for Splunk. This app works with the HCL BigFix Add-on for Splunk and utilizes the data ingested with this add-on in various dashboards to show the current state of BigFix.

If you do not see any data on this page, please check the current configuration of the "bigfix_index" macro and point it at the index where your BigFix data is.
